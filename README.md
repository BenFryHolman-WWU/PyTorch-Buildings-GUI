# PyTorch Buildings GUI

A graphical interface for building HVAC control using neural networks. Built with PNNL's NeuroMANCER library for differentiable predictive control.

## Installation

```bash
git clone https://github.com/BenFryHolman-WWU/PyTorch-Buildings-GUI.git
cd PyTorch-Buildings-GUI
chmod +x setup.sh
./setup.sh
source .venv/bin/activate
python src/main.py
```

Requires Python 3.12.x and Git.
